```mermaid
mindmap
  root((AWS AI))
    Artificial Intelligence
        Machine learning
        Deep Learning
        Generative AI
    AWS AI products
        Bedrock
        Sagemaker
    Classification Metrics
      Accuracy - correct predictions over total predictions = TP+TN/(TP+TN+FP+FN)
      Precision - how many positives are actually positive and not false alarms = Precision = TP/(TP+FP) - MEDICAL DIAGNOSIS
      Recall (sensitivity) - proportion of positive instances identified by the model = TP/(TP+FN) - FRAUD and security
      F1 Score - balance between precision and recall = 2x (precision x recall) / (precision + recall)
        
```
